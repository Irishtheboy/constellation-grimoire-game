# üöÄ GitHub Pages Deployment Guide

## Quick Setup

### 1. **Update package.json**
Replace `YOUR_USERNAME` in package.json with your GitHub username:
```json
"homepage": "https://YOUR_USERNAME.github.io/constellation-grimoire-game"
```

### 2. **Create GitHub Repository**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/constellation-grimoire-game.git
git push -u origin main
```

### 3. **Enable GitHub Pages**
1. Go to your repository on GitHub
2. Settings ‚Üí Pages
3. Source: "GitHub Actions"
4. Save

### 4. **Deploy**
Choose one method:

#### **Method A: Automatic (Recommended)**
- Push to main branch
- GitHub Actions will auto-deploy
- Check Actions tab for progress

#### **Method B: Manual**
```bash
npm run deploy
```

## üîß Configuration

### **Environment Variables**
For Firebase, add to GitHub repository secrets:
- `REACT_APP_FIREBASE_API_KEY`
- `REACT_APP_FIREBASE_AUTH_DOMAIN`
- etc.

### **Custom Domain (Optional)**
1. Add `CNAME` file to `public/` folder
2. Add your domain name inside
3. Configure DNS settings

## üêõ Troubleshooting

### **Build Fails**
- Check console for errors
- Ensure all imports are correct
- Verify Firebase config

### **Pages Not Loading**
- Check homepage URL in package.json
- Verify GitHub Pages is enabled
- Check Actions tab for deployment status

### **Firebase Issues**
- Add environment variables to GitHub secrets
- Update Firebase security rules
- Check console for auth errors
