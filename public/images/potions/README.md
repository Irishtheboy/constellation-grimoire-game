# Potion Images

Place your custom potion images in this folder:

## Required Files:
- `health.png` - Health potion
- `mana.png` - Mana potion
- `strength.png` - Strength boost potion
- `defense.png` - Defense boost potion
- `speed.png` - Speed boost potion
- `default.png` - Fallback potion

## Image Requirements:
- Format: PNG with transparency
- Size: 60x60 pixels recommended
- Style: Bottle/flask design
- Background: Transparent

## Animation:
Potions will show a sip animation with bubbles floating upward.
